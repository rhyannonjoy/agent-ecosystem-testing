The term \"machine learning\" was coined in 1959 by [Arthur Samuel](https://en.wikipedia.org/wiki/Arthur_Samuel_(computer_scientist)), an IBM computer scientist who wrote one of the first machine learning programs.[2][3] A more recent definition of machine learning by [Tom M. Mitchell](https://en.wikipedia.org/wiki/Tom_M._Mitchell) describes it as \"a computer program is said to learn from [experience](https://en.wikipedia.org/wiki/Experience) E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E\".[15] Machine learning is closely related to the fields of [statistics](https://en.wikipedia.org/wiki/Statistics), [optimisation](https://en.wikipedia.org/wiki/Mathematical_optimization), and [data mining](https://en.wikipedia.org/wiki/Data_mining), and is sometimes referred to as [predictive analytics](https://en.wikipedia.org/wiki/Predictive_analytics) or [statistical learning](https://en.wikipedia.org/wiki/Statistical_learning).\nRecently, generative artificial neural networks have been able to surpass many previous approaches in performance.[19][20] Machine learning algorithms have found applications in many fields, including [natural language processing](https://en.wikipedia.org/wiki/Natural_language_processing), [computer vision](https://en.wikipedia.org/wiki/Computer_vision), [speech recognition](https://en.wikipedia.org/wiki/Speech_recognition), [email filtering](https://en.wikipedia.org/wiki/Email_filtering), [self-driving cars](https://en.wikipedia.org/wiki/Self-driving_car), etc.[3][21]"

Mitchell](https://en.wikipedia.org/wiki/Tom_M._Mitchell) later provided a widely quoted, more formal definition of the algorithms studied in the machine learning field: \"A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E.\"[15] This definition is notable because it suggests the now well-known [bias-variance tradeoff](https://en.wikipedia.org/wiki/Bias%E2%80%93variance_tradeoff) in machine learning.[24] A more recent definition of machine learning, proposed by [Ethem Alpaydin](https://en.wikipedia.org/wiki/Ethem_Alpaydin), is as follows: \"Machine learning is programming computers to optimize a performance criterion using example data or past experience. ... The goal of machine learning is to develop methods and algorithms that can learn from data automatically, without human intervention or assistance.\"[49]\n[Andrey Kolmogorov](https://en.wikipedia.org/wiki/Andrey_Kolmogorov) and others developed the [theory of probability](https://en.wikipedia.org/wiki/Probability_theory) and [statistical inference](https://en.wikipedia.org/wiki/Statistical_inference) in the early 20th century, providing mathematical foundations for machine learning. [Alan Turing](https://en.wikipedia.org/wiki/Alan_Turing)'s seminal 1950 paper on [Computing Machinery and Intelligence](https://en.wikipedia.org/wiki/Computing_Machinery_and_Intelligence) proposed a [learning machine](https://en.wikipedia.org/wiki/Machine_learning) that could learn from experience, and introduced the [Turing test](https://en.wikipedia.org/wiki/Turing_test) to evaluate machine intelligence.[16]\nIn 1952, Arthur Samuel wrote the first computer learning program, which played the game of [checkers](https://en.wikipedia.org/wiki/Checkers), using [rote learning](https://en.wikipedia.org/wiki/Rote_learning) to improve its play.[5] In 1957, [Frank Rosenblatt](https://en.wikipedia.org/wiki/Frank_Rosenblatt) invented the [perceptron](https://en.wikipedia.org/wiki/Perceptron), the first artificial neural network, which could learn to classify simple patterns.[12] However, the limitations of the perceptron were highlighted in the 1969 book [Perceptrons](https://en.wikipedia.org/wiki/Perceptrons_(book)) by [Marvin Minsky](https://en.wikipedia.org/wiki/Marvin_Minsky) and [Seymour Papert](https://en.wikipedia.org/wiki/Seymour_Papert), which contributed to a decline in interest in neural network research and the onset of the first [AI winter](https://en.wikipedia.org/wiki/AI_winter).[23]\nIn the 1980s, the [backpropagation](https://en.wikipedia.org/wiki/Backpropagation) algorithm was popularized by [David Rumelhart](https://en.wikipedia.org/wiki/David_Rumelhart), [Geoffrey Hinton](https://en.wikipedia.org/wiki/Geoffrey_Hinton), and [Ronald Williams](https://en.wikipedia.org/wiki/Ronald_J._Williams) in 1986,[23] which enabled training of multi-layer neural networks. The [decision tree](https://en.wikipedia.org/wiki/Decision_tree_learning) algorithm was developed by [J. Ross Quinlan](https://en.wikipedia.org/wiki/J._Ross_Quinlan) in 1986, and [reinforcement learning](https://en.wikipedia.org/wiki/Reinforcement_learning) was formalized by [Richard Sutton](https://en.wikipedia.org/wiki/Richard_S._Sutton) and [Andrew Barto](https://en.wikipedia.org/wiki/Andrew_Barto) in the late 1980s.[23]\nIn the 1990s, machine learning shifted from a knowledge-driven approach to a data-driven approach. Scientists began creating programs for computers to analyze large amounts of data and draw conclusions – or \"learn\" – from the results. [Support vector machines](https://en.wikipedia.org/wiki/Support_vector_machine) (SVMs) became popular in the 1990s, and [random forests](https://en.wikipedia.org/wiki/Random_forest) were introduced by [Leo Breiman](https://en.wikipedia.org/wiki/Leo_Breiman) in 2001.[23]\nThe 2000s and 2010s saw the rise of [deep learning](https://en.wikipedia.org/wiki/Deep_learning), a subfield of machine learning based on [artificial neural networks](https://en.wikipedia.org/wiki/Artificial_neural_network) with many layers (hence \"deep\"). Key milestones include: [Yann LeCun](https://en.wikipedia.org/wiki/Yann_LeCun)'s [LeNet-5](https://en.wikipedia.org/wiki/LeNet) (1998) for handwritten digit recognition; the [ImageNet](https://en.wikipedia.org/wiki/ImageNet) competition win by [Alex Krizhevsky](https://en.wikipedia.org/wiki/Alex_Krizhevsky), [Ilya Sutskever](https://en.wikipedia.org/wiki/Ilya_Sutskever), and [Geoffrey Hinton](https://en.wikipedia.org/wiki/Geoffrey_Hinton)'s [AlexNet](https://en.wikipedia.org/wiki/AlexNet) in 2012 using [deep convolutional neural networks](https://en.wikipedia.org/wiki/Convolutional_neural_network); and [DeepMind](https://en.wikipedia.org/wiki/DeepMind)'s [AlphaGo](https://en.wikipedia.org/wiki/AlphaGo) defeating world champion [Lee Sedol](https://en.wikipedia.org/wiki/Lee_Sedol) at the game of [Go](https://en.wikipedia.org/wiki/Go_(game)) in 2016.[20] [Generative adversarial networks](https://en.wikipedia.org/wiki/Generative_adversarial_network) (GANs) were introduced by [Ian Goodfellow](https://en.wikipedia.org/wiki/Ian_Goodfellow) et al. in 2014.[19]"

Each training example has one or more inputs and a desired output, also known as a supervisory signal. Through iterative optimisation of an objective function, supervised learning algorithms learn a function that can be used to predict the output associated with new inputs.[44] An optimal function will allow the algorithm to correctly determine the output for inputs that were not a part of the training data. The algorithm gradually improves its predictions by adjusting its internal parameters. The learning process is guided by a [loss function](https://en.wikipedia.org/wiki/Loss_function) that measures the difference between the algorithm's predictions and the desired outputs.\nSupervised learning algorithms can be further divided into [classification](https://en.wikipedia.org/wiki/Classification_(machine_learning)) and [regression](https://en.wikipedia.org/wiki/Regression_analysis) algorithms. Classification algorithms are used when the output is a [discrete](https://en.wikipedia.org/wiki/Discrete_mathematics) value (e.g., a category or class), while regression algorithms are used when the output is a [continuous](https://en.wikipedia.org/wiki/Continuous_function) value (e.g., a number).\n[Decision trees](https://en.wikipedia.org/wiki/Decision_tree_learning), [support vector machines](https://en.wikipedia.org/wiki/Support_vector_machine) (SVMs), [k-nearest neighbors](https://en.wikipedia.org/wiki/K-nearest_neighbors_algorithm) (KNN), [linear regression](https://en.wikipedia.org/wiki/Linear_regression), [logistic regression](https://en.wikipedia.org/wiki/Logistic_regression), [naive Bayes classifiers](https://en.wikipedia.org/wiki/Naive_Bayes_classifier), and [artificial neural networks](https://en.wikipedia.org/wiki/Artificial_neural_network) are examples of widely used supervised learning algorithms."
