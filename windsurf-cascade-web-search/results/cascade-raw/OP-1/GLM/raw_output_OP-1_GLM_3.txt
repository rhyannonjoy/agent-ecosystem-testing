Mitchell provided a widely quoted, more formal definition of the algorithms studied in the machine learning field: \"A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E.\"[7] This definition is notable because it suggests the well-known formulation of the machine learning problem as an optimisation problem: defining a task as learning a target function that maps input data to output predictions, and seeking an algorithm that approximates this target function as closely as possible. This entails determining an objective function that provides a quantitative performance measure for the task, and seeking an algorithm that optimises this objective function. For example, a common objective function in classification tasks is cross-entropy loss, and the corresponding algorithm seeks to find model parameters that minimise this loss function.\n\nThis modern definition of the field reflects the shift from logic-based AI (so-called \"Good Old-Fashioned Artificial Intelligence\" (GOFAI)) to data-driven AI, focusing on building algorithms that can learn from data rather than relying on hand-coded rules.[8]"
