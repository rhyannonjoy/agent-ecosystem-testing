## What is a Search Index?

 [Image: Chevron Right Icon]

  
a MongoDB Search index is a data structure that categorizes data in an easily searchable format.
It maps terms with documents that contain those terms to enable fast retrieval
of documents at query-time.


You must configure a MongoDB Search index to query data using MongoDB Search. We recommend that you index the fields that you regularly use to sort or
filter your data.


To learn more, see [Supported Clients.](https://www.mongodb.com/docs/atlas/atlas-search/manage-indexes/#std-label-fts-manage-indexes)


In this section, you create a MongoDB Search index on the sample [movies](https://www.mongodb.com/docs/atlas/sample-data/sample-mflix/#std-label-mflix-movies) collection. You can create the MongoDB Search index in either an Atlas cluster or a deployment hosted on your local computer.

## What are Atlas Search queries?

 [Image: Chevron Right Icon]

  
MongoDB Search queries take the form of an [aggregation pipeline stage](https://www.mongodb.com/docs/manual/aggregation/). You use MongoDB Search primarily with the [$search](https://www.mongodb.com/docs/atlas/atlas-search/aggregation-stages/search/#mongodb-pipeline-pipe.-search) stage, which must be the first stage in the query pipeline. You can also use
this stage in conjunction with other stages in your pipeline.


When you run a MongoDB Search query, MongoDB Search uses the search index to locate and retrieve relevant data from the collection.

### Run Autocomplete and Partial Match Queries


You can use MongoDB Search to run autocomplete and partial queries that return results for partial words.
This is useful to retrieve results with increasing accuracy as more characters
are entered in your application's search field. You must index the fields as the MongoDB Search [autocomplete](https://www.mongodb.com/docs/atlas/atlas-search/field-types/autocomplete-type/#std-label-bson-data-types-autocomplete) type to use the [autocomplete](https://www.mongodb.com/docs/atlas/atlas-search/operators-collectors/autocomplete/#std-label-autocomplete-ref) operator.


To get started, see [How to Run Autocomplete and Partial Match MongoDB Search Queries.](https://www.mongodb.com/docs/atlas/atlas-search/tutorial/partial-match/#std-label-partial-match-tutorial)

### Group Search Results with Facets


You can use MongoDB Search to run facet queries that group search results by string, date, or numeric
values. You must create an index with a [facet definition](https://www.mongodb.com/docs/atlas/atlas-search/operators-collectors/facet/#std-label-fts-facet-definition) to use the [facet (MongoDB Search Operator)](https://www.mongodb.com/docs/atlas/atlas-search/operators-collectors/facet/#std-label-fts-facet-ref) collector.


To get started, see [How to Use Facets with MongoDB Search.](https://www.mongodb.com/docs/atlas/atlas-search/tutorial/facet-tutorial/#std-label-facet-tutorial)
